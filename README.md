# TanStack 

Это корпоративный сайт, предназначенный для презентации, публикации новостей и мероприятий, а также предоставления доступа к кабинету. 

## Технологический стек

- Frontend: React 18, TypeScript
- Роутинг: TanStack Router (файловая система маршрутов)
- Стилизация: SCSS Modules, CSS Variables
- Сборка: Vite
- Валидация форм: Formik, Yup
- Архитектура: Компонентный подход, BEM методология

## Особенности реализации

- Типобезопасная навигация между страницами
- Адаптивный дизайн для мобильных устройств, планшетов и десктопов
- Плавные анимации переходов между страницами
- Валидация форм входа и регистрации на стороне клиента
- Модульная структура стилей с использованием миксинов и переменных
- Семантическая верстка HTML5

## Структура проекта

src/
├── components/       # Переиспользуемые компоненты (Header, Footer)
├── routes/           # Страницы приложения (маршруты)
├── styles/           # Глобальные стили, переменные и миксины
├── main.tsx          # Точка входа приложения
└── router.tsx        # Конфигурация роутера

## Установка и запуск

1. Клонируйте репозиторий:
git clone <url-репозитория>

2. Перейдите в директорию проекта:
cd bako-tech

3. Установите зависимости:
npm install

4. Запустите сервер разработки:
npm run dev

5. Откройте браузер по адресу http://localhost:5173

## Сборка для продакшена

Для создания оптимизированной сборки выполните команду:
npm run build

Предпросмотр собранной версии:
npm run preview
